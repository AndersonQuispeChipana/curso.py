Operadores en Python
## Operadores unarios

Los operadores unarios actúan sobre un único valor. Son útiles para cambiar el signo de un número, negar expresiones lógicas o manipular bits.

Operador unario positivo (+)

Indica que el número es positivo. Generalmente no cambia el valor.

x = 8

print(+x)
## Operadores binarios

Trabajan con dos valores.

Ejemplos:

a + b
a > b
a and b
## Operador ternario

Trabaja con tres partes y sirve para hacer condiciones cortas.

Ejemplo:

"Mayor" if edad >= 18 else "Menor"
